# Nest Express Boilerplate

## Description

NestJS Express Boilerplate

## Prerequisite

- Node >= 16.x
- PM2
- PostgreSQL

## Additional Libraries

- @nestjs/config
- class-transformer
- class-validator
- rimraf
- @swc

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
